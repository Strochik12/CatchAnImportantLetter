# CatchAnImportantLetter

Иногда в корпоративной почте hse.ru попадаются письма, на которые надо отреагировать немедленно. Например, это сообщение о том, что на территории Покровского кампуса открыта запись на медосмотр для сотрудников. Или открыта запись на какой-то НИС или курс по выбору для студентов. Проблема в том, что надо успеть записаться, потому что даже через полчаса ссылка на запись неактуальна - все места уже заняты.
CatchAnImportantLetter - это программа, которая решает эту проблему. Она в реальном времени отслеживает ваш почтовый ящик, автоматически определяет срочные и важные письма и мгновенно уведомляет вас через выбранный канал связи: Telegram-бота, SMS или даже голосовой звонок с озвучкой содержимого.
